Projeto do modulo 02 - Programadores carioca.

Segundo trabalho do projeto módulo 02 - Projeto mesmo aonde o tempo foi limitadissimo, todos os participantes trabalharam para superar seus limites e conseguir entregar esse projeto maravilhoso. Projeto desenvolvido em um curso WebDev full-stack idealizado pelo Senac/Resilia para o Projeto Programadores Cariocas". “Programadores Cariocas” é um programa desenvolvido pela Prefeitura do Rio de Janeiro em parceria com o Senac RJ/Resilia, que visa formar mais de 1.000 jovens para atuar na área tecnológica e ingressar no mercado de trabalho com qualificação. Projeto mesmo aonde o tempo foi limitadissimo, todos os participantes trabalharam para superar seus limites e conseguir entrar esse projeto maravilhoso. Todo o conteúdo do site é fictício.
